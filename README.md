# DNCL2

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A collection of sample programs for the high school "Information I" subject, demonstrating common test representation.

## Features
- Examples of program representation for the "Information I" subject in high school.
- Includes variable definitions, string handling, assignment, arithmetic operations, comparison operations, logical operations, functions, conditional statements, and loops.
- Provides a reference to the overview of the "Information" subject in the Common Test for University Admissions.

## Usage
This project serves as a reference for program representation used in the "Information I" common test.
